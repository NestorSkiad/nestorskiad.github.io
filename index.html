Hello!

I'm Nestor, a back-end web developer with a specialization in very high performance web applications. I can write a mean SQL query and I parallelise with ease. When I'm not on the clock, I make small coding projects, I'm a road cyclist, I do photography and I make videos for YouTube.

Projects

  • BSYearbot: Social media bot that, when mentioned, tells the user the current year, along with a fun PG-rated insult. Written in Python, deployed on a Raspberry Pi. Currently running on @whatyearisit.bsky.social .
  • Nearest Color GUI: For a given color that the user inputs, we calculate its Euclidean distance from each of the ~850 named colors we've got and find the minimum. Written in Rust.
  • BattleGo: Distributed REST API for playing the game Battleship. Includes matchmaking logic, session management, in-memory storage of match state. Written in Go, uses PosgreSQL for its database.
  • RPiPi: Displays consecutive digits of Pi on a seven-segment display connected to a Raspberry Pi zero. It's resilient, picking up from where it left off if the Raspberry Pi loses connection or power. Written in Rust (with no-std).
